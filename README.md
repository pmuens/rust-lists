# Rust Lists

Code written while following ["Learn Rust by writing Entirely Too Many Linked Lists"](https://rust-unofficial.github.io/too-many-lists).

## Useful Commands

```sh
nix-shell

cargo init
cargo new <name>

cargo build [--bin <binary>]
cargo clean
cargo test
cargo bench

cargo update
cargo add <name>
cargo search <name>

cargo run [--bin <binary>] [-- arg]

cargo doc [--open]
cargo fix
cargo fmt
cargo clippy
cargo check

cargo install <name>
cargo uninstall <name>
```
